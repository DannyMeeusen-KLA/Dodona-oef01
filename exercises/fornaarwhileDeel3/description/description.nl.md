De gebruiker dienteen zin van minstens twee woorden in te geven.  
Het programma drukt de woorden lijn per lijn af.  
Je krijgt de oplossing cadeau waarbij gebruik gemaakt werd van een for.  

Vervang in het programma de for door een while zodat je hetzelfde resultaat krijgt.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> Dit is een zin.
Dit
is
een
zin.
>>> Remco Evenepoel!
Remco
Evenepoel!
>>> Wie won de wedstrijd tussen Antwerp en Beerschot?
Wie
won
de
wedstrijd
tussen
Antwerp
en
Beerschot?
```
