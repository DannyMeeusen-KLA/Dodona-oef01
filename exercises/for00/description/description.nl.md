Schrijf een programma dat  
- de alfabetische lijst met de Vlaamse provincies maakt (je krijgt deze instructie cadeau, zie onderaan)
- de gebruiker de kans geeft om een woord in te tikken,  
- de gebruiker de kans geeft om een exponent *n* in te tikken.    

Het programma drukt vervolgens af, telkens op een nieuwe lijn:
- alle provincies uit de lijst  
- letters uit het woord
- de eerste *n* volkomen kwadraten, dus 1^2^, 2^2^, 3^2^, ... n^2^  

Je kan deze instructie knippen en plakken om de lijst met provincies te vullen:
provincies = ['Antwerpen', 'Limburg', 'Oost-Vlaanderen', 'Vlaams-Brabant', 'West-Vlaanderen']
