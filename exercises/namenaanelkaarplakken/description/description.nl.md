Schrijf een programma waarbij de gebruiker twee strings kan ingeven: zijn voornaam en zijn familienaam.  
Druk daarna de voornaam en de familienaam af, gescheiden door een spatie.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> Jack
>>> Lloyd
Jack Lloyd
>>> Edward
>>> Cloetens
Edward Cloetens
>>> Ramazan resul
>>> Ozlu
Ramazan resul Ozlu
>>> Gaël
>>> Michaud
Gaël Michaud
>>> Mia
>>> Adams
Mia Adams
>>> Julie
>>> Cantillon
Julie Cantillon
```
