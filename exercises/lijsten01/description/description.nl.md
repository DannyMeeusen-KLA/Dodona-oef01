Schrijf een programma waarbij de gebruiker in één zin 3 getallen ingeeft, gescheiden door een spatie.  
Steek deze getallen als een integer in een tabel (niet als string!).  
Druk de tabel af.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 5 8 11
[5, 8, 11]
>>> 16 0 -5
[16, 0, -5]
>>> 123 567 -789
[123, 567, -789]
```
