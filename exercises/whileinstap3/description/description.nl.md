De gebruiker dient een tekst in te geven (géén getal!).  
Vervolgens wordt deze tekst steeds achter elkaar geplakt, tot de resultaatstring een lengte van meer dan 20 heeft.  
Deze resultaatstring drukken we af.  
  
Let op: we voegen aan de resultaatstring steeds de string toe die door de gebruiker wordt ingegeven. Je zal deze waarde dus in een aparte variabele moeten bewaren!    

### Voorbeeld

```console?lang=python&prompt=>>>
>>> abcdef
abcdefabcdefabcdefabcdef
>>> Driemaal
DriemaalDriemaalDriemaal
>>> Negentienletterwoord
NegentienletterwoordNegentienletterwoord
>>> Vierentwintigletterwoord
Vierentwintigletterwoord
>>> Vijfx
VijfxVijfxVijfxVijfxVijfx
```
