De gebruiker geeft een tekst in.    
Het programma drukt, lijn per lijn, elke individuele letter van de tekst af.    
Je krijgt de oplossing cadeau waarbij gebruik gemaakt werd van een for.  

Vervang in het programma de for door een while zodat je hetzelfde resultaat krijgt.
