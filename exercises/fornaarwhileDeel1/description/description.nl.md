De gebruiker dient, op twee verschillende lijnen, de boven- en ondergrens  
van een gesloten interval in te geven. Het programma drukt, lijn per lijn,  
de gehele getallen uit dit interval af.  
Je krijgt de oplossing cadeau waarbij gebruik gemaakt werd van een for-range.  

Vervang in het programma de for door een while zodat je hetzelfde resultaat krijgt.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 5
>>> 8
5
6
7
8
>>> -1
>>> 1
-1
0
1
>>> -6
>>> -2
-6
-5
-4
-3
-2
>>> 0
>>> 2
0
1
2
>>> 100
>>> 105
100
101
102
103
104
105
```
