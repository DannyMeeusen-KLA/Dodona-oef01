Schrijf een programma waarbij twee gehele getallen worden ingegeven.  
Als minstens één van de twee getallen strikt negatief is (d.w.z. kleiner dan 0), dan drukt het programma de tekst NEGATIEF af.  
Het getal 0 wordt als een positief getal beschouwd. Als twee keer het getal 0 wordt ingegeven, dan drukt het programma dus de tekst POSITIEF af.

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 15
>>> 23
POSITIEF
>>> 13
>>> -15
NEGATIEF
>>> 0
>>> 0
POSITIEF
>>> -9
>>> 8
NEGATIEF
>>> 8
>>> 0
POSITIEF
```
