Schrijf een programma waarbij de gebruiker een natuurlijk getal kan ingeven.  
Het programma berekent de faculteit van dit getal.  

De faculteit van een natuurlijk getal is een wiskundige uitdrukking die heel vaak in de combinatieleer en het kansrekenen
gebruikt wordt. De faculteit van een natuurlijk getal *n* wordt als volgt berekend:  
  
*n! = n . (n-1) . (n-2) ... 1*
  
Speciale gevallen:  

*1! = 1*  
*0! = 1*

PS: De faculteit van een natuurlijk getal wordt al snel erg groot. Zo is *20! = 2 432 902 008 176 640 000* (meer dan 2 triljoen)

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 5
120
>>> 8
40320
>>> 10
3628800
>>> 1
1
>>> 0
1
```
